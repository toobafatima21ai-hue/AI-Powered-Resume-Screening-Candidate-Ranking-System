# AI-Powered Resume Screening & Candidate Ranking System

## Overview

The AI-Powered Resume Screening & Candidate Ranking System is a Streamlit-based application designed to automate and enhance the recruitment process. The system analyzes resumes, extracts relevant candidate information, compares qualifications against job requirements, and ranks candidates based on their suitability for a given role.

By leveraging Natural Language Processing (NLP), semantic similarity matching, and Google's Gemini AI, the application provides recruiters with intelligent insights, skill-gap analysis, candidate summaries, and interview recommendations.

---

## Key Features

### Resume Parsing

* Extracts candidate information from PDF resumes.
* Identifies skills, education, experience, and other relevant details.
* Supports automated resume analysis with minimal manual effort.

### Job Description Analysis

* Processes job descriptions and extracts required skills and qualifications.
* Creates a structured representation of job requirements.

### Candidate Ranking

* Compares resumes against job requirements.
* Calculates candidate-job match scores.
* Ranks candidates according to relevance and suitability.

### AI-Powered Insights

* Generates professional candidate summaries using Gemini AI.
* Performs skill-gap analysis.
* Creates targeted interview questions.
* Provides personalized resume improvement suggestions.

### Interactive Dashboard

* Built with Streamlit for an intuitive user experience.
* Resume upload interface.
* Candidate comparison tools.
* Visual ranking and evaluation workflow.

## Workflow

1. Enter a job title and job description.
2. Upload one or more candidate resumes in PDF format.
3. Analyze and rank candidates automatically.
4. Review AI-generated summaries and recommendations.
5. Compare candidates and identify skill gaps.
6. Export results for recruitment decision-making.
